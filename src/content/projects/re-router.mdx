## Overview

RE-Router automates the decision-making process in binary analysis workflows. Instead of manually choosing which tools and techniques to apply, RE-Router uses intelligent decision trees to route analysis tasks to the appropriate tooling.

## How It Works

Given a binary, RE-Router:

1. Performs initial triage (file type, architecture, protections)
2. Routes to appropriate analysis pipelines based on characteristics
3. Orchestrates tool execution and aggregates results
4. Presents findings in a unified format

## Use Cases

- **Malware triage** — quickly classify and route samples
- **Vulnerability research** — automate repetitive analysis steps
- **CTF challenges** — rapid binary classification and initial analysis
